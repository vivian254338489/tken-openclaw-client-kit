## Community Post Draft

```text
I made a small OpenClaw-style provider config kit for OpenAI-compatible API endpoints.

It generates a local provider config with one base URL, a low-cost default model route, and a premium model route. It is meant for people who want to test model routing without hand-editing provider JSON.

Disclosure: I work on TKEN-related tooling. The default base URL is TKEN, but you can change it to any OpenAI-compatible provider.
```
